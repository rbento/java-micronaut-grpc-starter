# java-micronaut-grpc-starter

A starter for gRPC/Java projects backed by Micronaut.

### Requirements

- [Gradle](https://gradle.org/releases/) 8.2.1
- [JDK 17](https://www.azul.com/downloads/?package=jdk#zulu) from Azul

### Build

```bash
./gradlew clean build
```

### Run

```bash
./gradlew :server:run
```
